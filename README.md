> Get a country VAT rate. ISO 3166-1 compliant.

## Install

```bash
$ npm install country-vat --save
```

## Usage

```js
const countryVat = require('country-vat')

countryVat('ES') // => 0.21
countryVat('ESP') // => 0.21
countryVat(724) // => 0.21
countryVat('Spain') // => 0.21
```
